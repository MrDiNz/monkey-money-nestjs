สร้าง API สำหรับ CRUD ข้อมูลกู้ยืม โดยมีข้อมูลดังนี้
1.ผู้กู้ และ ผู้ค้ำประกัน มีข้อมูล ชื่อ สกุล บ้านเลขที่ หมู่ที่ ตำบล อำเภอ จังหวัด รหัสบัตรประชาชน เบอร์ lat long อาชีพ
2 ชิดรถ มีข้อมูล รุ่น ประเภท สี ปีจดทะเบียน เลขตัวถัง เลขเครื่องยนต์ ทะเบียน(ทั้งตัวเลขและจังหวัด)

โดยทั้งหมด เก็บแยก text field ให้พนักงานสามารถกรอกข้อมูลเองได้
แล้วจะทำข้อมูลตารางงวดชำระเงินภายหลัง feature นี้จบ
